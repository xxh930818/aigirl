# AI Girl - Live2D

Generated with [Project Starter CLI](https://github.com/xxh930818/project-starter).

## 特性

- React 18 + TypeScript
- Vite 构建工具
- Live2D 模型展示支持
- 美少女主题样式

## Live2D 模型设置

1. 在 `public/models` 目录下放置你的 Live2D 模型文件
2. 修改 `src/components/Live2D.tsx` 中的模型路径
3. 支持的模型格式：.moc3, .moc

## 开始开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 预览构建

```bash
npm run preview
```
